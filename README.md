# 全能模型中心

AI 模型交互平台，支持多种AI服务商的统一管理、对话测试和批量处理。

## 功能特性

- 🔧 **服务商管理**: 支持 OpenAI、Anthropic、Google、Azure 等主流AI服务商
- 💬 **AI 演练场**: 实时对话界面，支持多服务商切换
- 📁 **云盘批量处理**: 文件上传和批量AI任务处理
- ☁️ **云端配置**: 基于 JSONBin 的跨设备配置同步

## 使用方法

1. 在服务商管理中添加您的AI API配置
2. 在演练场中选择服务商进行对话测试
3. 在云盘页面上传文件进行批量处理

## 技术栈

- React + TypeScript + Vite
- Tailwind CSS + Lucide Icons
- JSONBin API (配置存储)
